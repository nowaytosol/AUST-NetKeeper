# AUST 校园网守护者

安徽理工大学（AUST）校园网自动登录与断线重连守护工具。基于 Dr.COM 网关协议，实时监控网络状态，掉线后毫秒级自动重连。

## 功能特性

- **自动登录** — 一键填写账号密码，向 Dr.COM 网关发起认证
- **断线守护** — 后台持续监控网络状态，掉线后自动重连
- **多运营商支持** — 电信（@aust）、移动（@cmcc）、联通（@unicom）、教职工（@jzg）
- **系统托盘** — 最小化到托盘运行，断线/重连气泡通知
- **开机自启** — 注册表级静默自启，开机即守护
- **密码加密存储** — 使用 Fernet（AES-128）加密本地配置中的密码
- **连续失败自停** — 连续 3 次登录失败自动停止，避免无效重试

## 环境要求

- Windows 10/11
- Python 3.8+（或使用打包后的 exe）
- 校园网有线/Wi-Fi 已连接（网关 `10.255.0.19` 可达）

## 安装与运行

### 从源码运行

```bash
# 安装依赖
pip install requests cryptography pystray pillow

# 运行
python AUST_NetKeeper.py
```

### 打包为 exe

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile AUST_NetKeeper.py
```

打包产物在 `dist/AUST_NetKeeper.exe`，可直接分发给同学使用。

## 使用说明

1. 启动后填写**学号**（纯数字）和**密码**
2. 选择对应的**网络出口**（不确定选"学生电信出口"）
3. 点击「▶ 开始运行」
4. 程序会持续守护，窗口可最小化到托盘

### 高级设置

| 参数 | 说明 | 默认值 | 建议范围 |
|---|---|---|---|
| 重连间隔 | 每次重连尝试之间的冷却秒数 | 5s | 3-10s |
| 检测间隔 | 两次网络状态探测之间的间隔 | 30s | 15-60s |

### 开机自启

勾选「开机静默启动」后，程序会写入注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`，开机后自动隐藏到托盘运行。

## 技术原理

程序通过两个 Dr.COM 网关 HTTP 接口工作：

### 在线状态检测

```
GET http://10.255.0.19/drcom/chkstatus?callback=dr1002
```

网关返回 JSONP 格式响应，`result:1` 表示已在线，`result:0` 表示未登录。

### 登录认证

```
GET http://10.255.0.19/drcom/login
    ?callback=dr1003
    &DDDDD=<学号>@<运营商后缀>
    &upass=<密码>
    &0MKKey=123456
    &R1=0&R3=0&R6=0&para=00&v6ip=&v=3404
```

参数来源于学校网关登录页面的 JavaScript 源码（Dr.COM WebLogin v3.3.2）。`0MKKey=123456` 和 `v=3404` 是 Dr.COM 协议的静态占位符，不随固件更新变化。

### 守护循环

```
while 未停止:
    在线状态 = 探测 chkstatus
    if 断线:
        发起 login 认证
        if 连续失败 ≥ 3 次:
            自动停止，释放 UI
    sleep(可中断的检测间隔)
```

## 常见问题

**Q: 启动后提示"登录失败"？**

检查学号和密码是否正确。注意密码是明文传输给网关的，确保在校园网环境下使用。

**Q: 托盘图标不显示？**

部分精简版 Windows 可能缺少系统托盘支持。尝试通过菜单「显示窗口」恢复主界面。

**Q: 开机自启不生效？**

安全软件可能拦截了注册表写入。手动将 exe 快捷方式放入 `shell:startup` 文件夹作为替代方案。

**Q: 程序移动位置后开机自启失效？**

注册表记录的是绝对路径。移动后需取消并重新勾选「开机静默启动」来更新路径。

## 配置文件说明

| 文件 | 用途 |
|---|---|
| `aust_net_config.json` | 用户配置（密码已加密） |
| `.aust_net.key` | Fernet 加密密钥（首次运行自动生成） |
| `aust_login.log` | 运行日志（自动轮转，最大 5MB × 3 份） |

配置文件与 exe 放在同一目录下，换机器时一起拷贝即可（密钥文件也要带）。
