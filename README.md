# AUST-NetKeeper

一款专为**安徽理工大学** 设计的校园网自动登录/断线重连工具，基于 Python 开发，自带可视化 GUI，支持多运营商出口、开机自启、密码加密存储等核心功能，彻底解决校园网频繁掉线的问题。

## 📌 核心特性
- 🖥 **可视化操作界面**：基于 Tkinter 开发，无需命令行，小白也能轻松上手；
- 🔌 **多出口适配**：支持学生电信/移动/联通、教职工专属出口，严格匹配校园网网关规则；
- 🔒 **密码加密存储**：采用 Fernet 对称加密算法保存密码，避免明文泄露风险；
- 🔄 **自动重连守护**：自定义检测/重连间隔，断线后自动触发重连，无需手动操作；
- 🚀 **开机静默自启**：写入 Windows 注册表，开机自动后台运行（最小化到系统托盘）；
- 📝 **实时日志监控**：界面展示运行日志，同时生成日志文件，可追溯连接状态；
- ⚡ **轻量高效**：毫秒级网关状态检测（基于 Dr.COM 官方 API），低资源占用。

## 🛠 环境要求
- Python 3.8 及以上版本（若使用打包后的 exe，无需安装 Python）；
- Windows 系统（依赖 Windows 注册表实现开机自启、系统托盘仅适配 Windows）；
- 校园网内网环境（仅适用于安徽理工大学校园网）。

## 🚀 快速开始
### 方式1：直接运行源码
1. 克隆本仓库：

2. 安装依赖：

pip install requests cryptography pillow pystray

3. 运行程序：

python AUST_NetKeeper.py

方式2：打包成 EXE（可选）

若需无 Python 环境运行，可使用 pyinstaller 打包：

pip install pyinstaller requests cryptography pystray pillow
pyinstaller --noconsole --onefile AUST_NetKeeper.py

打包后可在 dist 目录找到 exe 文件，双击即可运行。

📖 使用说明

基础操作

1. 运行程序后，填写纯数字学号和校园网密码；

2. 选择对应的网络出口（默认：学生电信出口）；

3. （可选）调整「重连间隔」（1-60秒）、「检测间隔」（5-300秒）；

4. 勾选「开机静默启动」可实现开机自动运行（最小化到托盘）；

5. 点击「▶ 开始运行」，程序将自动检测网络状态，断线后自动重连。

高级操作

- 最小化到托盘：点击菜单 → 最小化到托盘，程序将隐藏窗口，在系统托盘运行；

- 恢复窗口：点击系统托盘图标 → 打开主控制面板；

- 清空日志：点击「🗑 清空日志」可清除界面日志（日志文件仍保留）；

- 停止运行：点击「⏹ 停止运行」可暂停监控，恢复输入框编辑状态。

日志与配置

- 运行日志：自动生成 aust_login.log 文件，记录所有连接/重连/异常信息；

- 配置文件：aust_net_config.json 保存账号、出口、间隔等配置（密码加密存储）；

- 加密密钥：.aust_net.key 为密码加密密钥，删除后将重新生成（原保存的密码将失效）。

📂 项目结构

AUST-NetKeeper/

⚠ 注意事项

1. 仅适用于安徽理工大学校园网 Dr.COM 网关，其他院校暂不支持；

2. 开机自启功能可能被安全软件拦截，若失效请手动放行；

3. 若频繁重连失败，请检查学号/密码/出口是否正确，或校园网是否欠费；

4. 请勿修改 aust_net_config.json 和 .aust_net.key 文件，否则可能导致配置失效。
